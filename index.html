PREREQUISITES

간단한 사전 세팅

✏️

① Cursor

Claude Code를 실행할 에디터입니다. 이걸 먼저 설치하세요.

cursor.sh에서 다운로드 →

무료 플랜으로 충분합니다. 유료 결제는 필요 없어요.

📦

② Git

교육 자료를 다운받는 도구입니다.

Cursor 터미널에서 아래 명령어를 입력하세요:

xcode-select --install

팝업이 뜨면 "설치" 클릭. 이미 설치됐다면 에러가 나오는데 무시해도 됩니다.

git-scm.com에서 다운로드 →

설치 시 모든 옵션은 기본값(Next)으로 진행하세요.

중요: 설치 후 반드시 Cursor를 재시작하세요.

⚡

③ Node.js

Claude Code가 돌아가는 엔진입니다.

nodejs.org에서 LTS 다운로드 →

반드시 LTS(Long Term Support) 버전을 설치하세요.

Windows: 설치 중 "Add to PATH" 옵션이 체크되어 있는지 확인하세요.

🤖

④ Claude 계정

Pro($20/월) 이상 구독이 필요합니다.

claude.ai에서 가입 →

Pro 플랜 구독 후 Claude Code 사용이 가능합니다.

🪟 Windows 사용자를 위한 추가 안내

실제 교육에서 가장 많이 발생한 문제들과 해결 방법입니다.

❌ 'npm' is not recognized
Node.js 설치 후 Cursor(또는 터미널)를 완전히 종료 → 재시작해야 합니다. 설치만으로는 PATH가 반영되지 않아요.
❌ 'git' is not recognized
Git 설치 후 마찬가지로 Cursor 재시작 필요. 그래도 안 되면 Windows 검색 → "환경 변수 편집" → Path에 C:\Program Files\Git\cmd 추가.
❌ PowerShell 실행 정책 오류
PowerShell을 관리자 권한으로 열고 아래 명령어 실행:
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
❌ npm install 권한 오류 (EACCES / EPERM)
Cursor 터미널 대신, Windows 검색 → "cmd" → 관리자 권한으로 실행 후 npm install 재시도.
💡 꿀팁: 그래도 안 되면?
컴퓨터를 완전히 재부팅하세요. Windows 환경 변수 문제의 90%는 재부팅으로 해결됩니다.

Git 설치 확인

터미널에 아래 명령어를 입력하세요:

xcode-select --install

팝업이 뜨면 "설치" 클릭. 이미 설치됐다면 에러가 나오는데 무시해도 됩니다.

Node.js 설치

nodejs.org에서 LTS 버전을 다운로드 후 설치하세요.

Windows: 설치 중 "Add to PATH" 옵션이 체크되어 있는지 확인하세요.

5

Claude Code 설치

터미널에 아래 명령어를 복사 → 붙여넣기 하세요:

npm install -g @anthropic-ai/claude-code
bash
$ npm install -g @anthropic-ai/claude-code
added 1 package in 30s
✓ Claude Code installed successfully
1~2분 걸릴 수 있습니다. 화면에 텍스트가 쭉 지나가는 건 정상이에요.
✓ claude --version 입력 → 버전 번호가 뜨면 성공
6

Claude 로그인

터미널에서 claude를 입력하면 로그인 화면이 나옵니다.

claude
브라우저가 자동으로 열립니다. Claude 계정으로 로그인하세요.
✓ 로그인 완료 후 터미널에 Claude가 인사하면 성공
7

교육 자료 다운로드 + 시작

교육 자료를 다운받고, Claude Code를 한 번에 실행합니다.

git clone https://github.com/exemchs/exem-ai-curriculum-program.git && cd exem-ai-curriculum-program && claude

방법 1) 터미널에 입력:

source ~/.zshrc

Windows라면: source ~/.bashrc 또는 터미널 재시작

방법 2) Claude Code 재설치:

npm install -g @anthropic-ai/claude-code

방법 3) Cursor를 완전히 종료(Cmd+Q / Alt+F4) 후 다시 실행
