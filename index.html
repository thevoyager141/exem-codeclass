It's Time for
Claude Code

Claude Code를 이용하면 앱에서 채팅하던 때와는
수준이 다른 결과물, 자동화가 실제로 가능해집니다.

Scroll

'나중에 공부해야지' 생각했던 Claude Code,
지금이 기회입니다.

일일이 앱을 켜서 프롬프트를 입력하고, 파일을 첨부하고, 결과물을 수정해야 했던 불편함.
코드 에디터로 Claude Code를 활용하면 훨씬 간편하게, 강력하게 성과를 낼 수 있습니다.
필요한 건 딱 3가지:
├── Cursor  (무료 코드 에디터 — 코딩 안 합니다)
├── Claude Code  (Cursor 안에서 돌아가는 AI 도구)
└── 5일간의 호기심
그래서 우리가 배우는 건 코딩이 아닙니다.
AI가 '내 생각대로' 계획을 세우고, 자료를 분석하고, 일할 수 있도록
시스템을 만드는 법을 배웁니다.

5분이면 준비 끝

아래 7단계를 순서대로 따라하세요.

1

Cursor 에디터 설치

Claude Code를 실행하는 '공간'입니다. cursor.sh에서 다운로드 후 설치하세요.

Cursor 유료 결제는 걱정 안 하셔도 됩니다. 무료로 충분해요.
✓ Cursor를 실행했을 때 에디터 화면이 뜨면 성공
2

Cursor에서 터미널 열기

터미널 단축키
Mac: Cmd + J   |   Windows: Ctrl + J
✓ 화면 하단에 검은 배경의 터미널 창이 뜨면 성공
3

Claude Code 설치

터미널에 아래 명령어를 복사 → 붙여넣기 하세요:

bash

✓ Claude Code installed successfully
curl -fsSL https://claude.ai/install.sh | bash
1~2분 걸릴 수 있습니다. 화면에 텍스트가 쭉 지나가는 건 정상이에요.
✓ claude --version 입력 → 버전 번호가 뜨면 성공
4

교육 자료 다운로드

git clone https://github.com/exemchs/exem-ai-curriculum-program.git
✓ "done" 또는 다운로드 완료 메시지가 뜨면 성공
5

프로젝트 폴더 열기

Cursor 상단 메뉴 → File → Open Folder → exem-ai-curriculum-program 선택

폴더 위치: Mac: ~/exem-ai-curriculum-program | Windows: C:\Users\내이름\exem-ai-curriculum-program
✓ Cursor 왼쪽에 CLAUDE.md, README.md 등 파일이 보이면 성공
6

터미널을 새로 열기

프로젝트 폴더 안에서 터미널이 실행되어야 합니다.

왜 다시 열어야 하나요? 프로젝트 폴더 안에서 터미널이 실행되어야 Claude Code가 교육 자료를 인식합니다.
✓ 터미널 경로에 exem-ai-curriculum-program이 보이면 성공

⚠️ 이어하기가 안 될 때

증상: command not found: claude

Cursor에서 새 터미널을 열면 발생할 수 있어요.

방법 1) 터미널에 입력:

source ~/.zshrc

Windows라면: source ~/.bashrc

방법 2) Claude Code 재설치:

curl -fsSL https://claude.ai/install.sh | bash

방법 3) Cursor를 완전히 종료(Cmd+Q / Alt+F4) 후 다시 실행

증상: "이어서 해줘"에 엉뚱한 응답

원하는 Day를 직접 지정하세요:

/day3

자주 묻는 질문

"터미널"이 뭔가요?+
컴퓨터에게 텍스트로 명령을 내리는 창입니다. 마우스로 클릭하는 대신, 글자를 입력해서 지시하는 방식이에요. 이 교육에서는 복사+붙여넣기만 하면 되니 걱정 마세요.
Claude Code랑 Claude 앱(채팅)은 뭐가 다른가요?+
Claude 앱 = 웹 브라우저에서 대화하는 방식. Claude Code = 코드 에디터(Cursor) 안에서 실행하는 방식. Claude Code는 파일을 직접 만들고, 도구를 연결하고, 자동화할 수 있어서 "업무 자동화"가 가능합니다.
Cursor 유료 결제가 필요한가요?+
아니요, 무료 플랜으로 충분합니다. Cursor는 Claude Code를 실행하는 "공간"으로만 사용하고, AI 기능은 Claude Code가 담당합니다.
프로그래밍을 전혀 몰라도 되나요?+
네! 이 프로그램은 비전공자를 위해 설계되었습니다. 코드를 직접 짜는 게 아니라, Claude Code에게 "이렇게 해줘"라고 시키는 법을 배우는 겁니다.
명령어를 입력했는데 한참 동안 아무것도 안 돼요.+
정상입니다! 설치나 다운로드 중에는 1~2분 걸릴 수 있습니다. 화면에 텍스트가 올라가거나 커서가 깜빡이면 작동 중인 겁니다. 5분 이상 아무 반응이 없을 때만 걱정하세요.
"command not found: claude" 에러가 떠요.+
Cursor에서 터미널을 새로 열면 종종 발생하는 문제입니다. 터미널에 source ~/.zshrc (Mac) 또는 source ~/.bashrc (Windows)를 입력해보세요. 그래도 안 되면 Cursor를 완전히 종료 후 다시 실행하세요.
